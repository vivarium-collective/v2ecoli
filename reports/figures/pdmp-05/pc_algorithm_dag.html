PC-algorithm causal DAG

illustrative phase5-output

The PC algorithm (Peter & Clark, 1991) is the canonical constraint-based causal discovery method. From a conditional-independence oracle (or finite-sample test) it recovers the CPDAG — the set of DAGs consistent with the observed independencies. Shown: illustrative recovered DAG over 8 WCM variables. Real Phase 5 output runs against the Phase 0 reference ensemble + interventional data from synthetic gene knockouts.
