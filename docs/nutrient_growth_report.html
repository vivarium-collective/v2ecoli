Glucose uptake kinetics

With Michaelis-Menten enabled in metabolic_kinetics.py, the glucose import bound is computed every step from boundary.external.GLC instead of being fixed by a media lookup. Parameters: v_max = 20.0 mmol/gDCW/h, K_m = 0.01 mM.

Work in progress: environment depletion feedback is not yet wired — boundary.external.GLC stays at its initial media value, so the left panel above is effectively flat and the uptake bound (right panel) stays saturated. The next commit on the nutrient-growth branch will subtract cellular uptake flux from the external pool so the model naturally transitions into stationary phase as glucose runs out.

What this report does not cover yet

  • Environment depletion: cell uptake currently does not subtract from boundary.external. Next commit.
  • Non-glucose carbon sources: glycerol, gluconate, lactate. Requires media recipes + per-carbon v_max/K_m.
  • Ion stress: Na⁺, Mg²⁺ gradients from Caglar's figure 2B/C.
  • ppGpp-driven transition: mechanistic stationary- phase entry via the existing ppgpp_initiation pathway.
